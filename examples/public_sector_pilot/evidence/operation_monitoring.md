# Operation Monitoring Procedure

Provider instructions: use the assistant only for its intended purpose of
public-service information support, and follow provider limitations.
Monitoring plan: review behavior traces, release gates, resident feedback, risk
events, and serious incident reports after each model or prompt change.
Responsible operator: assigned pilot operator under the Public Sector AI Risk
Owner.
Risk review trigger: any release gate review status, missing evidence, incident,
or provider model update triggers review before wider deployment.
